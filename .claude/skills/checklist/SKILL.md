# Checklist 스킬

`docs/planning/CHECKLIST.md`를 읽어 현재 Phase 진행 상황을 요약하고, 다음 작업을 안내한다.

## 트리거

- "체크리스트 확인해줘 / 보여줘"
- "다음 작업이 뭐야"
- "~~ 체크해줘 / 완료 표시해줘"
- "체크리스트 업데이트해줘"

## 실행 흐름

### 조회 모드 (확인/보여줘/다음 작업)

1. `docs/planning/CHECKLIST.md` Read
2. 현재 Phase 섹션에서 미완료(`[ ]`) 항목 추출
3. 아래 포맷으로 출력:

```
## 현재 Phase: {Phase명}

### 완료
- [x] 항목 (간결하게)
...

### 미완료 (다음 작업 순서)
1. [ ] 항목
2. [ ] 항목
...

→ 바로 시작할 항목: {첫 번째 미완료 항목}
```

### 업데이트 모드 (체크해줘/완료 표시)

1. 사용자가 지정한 항목을 `docs/planning/CHECKLIST.md`에서 찾는다
2. `[ ]` → `[x]` 변경 (Edit 툴 사용)
3. "마지막 업데이트" 날짜를 오늘로 갱신
4. 변경 사항 요약 출력

## 출력 규칙

- 전체 체크리스트를 그대로 출력하지 않는다 — 현재 Phase 섹션만
- 완료 항목은 개수만 표시, 미완료 항목만 목록으로 표시
- 다음 작업은 굵게 강조
